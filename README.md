
## Что сделано

### Вeрстка
- Адапивная вeрстка от 1920px до 320px
- Резиновое масштабирование без жeстких брейкпоинтов (используется clamp и fluid-функция в SCSS)
- Стили по методологии BEM

### Функционал
- Поиск по названию курса в реальном времени (без перезагрузки)
- Фильтрация по категориям (Marketing, Management, HR & Recruiting, Design, Development)
- Карточки группируются по категориям автоматически
- Если выбрана конкретная категория - показываются только курсы этой категории
- Если выбрано "all" - показываются все курсы, сгруппированные по категориям

### Технические детали
- Чистый JavaScript, никаких фреймворков
- Данные в `course-data.js` - массив курсов с полями: sphere, name, price, author, img
- Код разбит на небольшие функции, всe понятно и просто
- Используется data-атрибуты для связи JS с HTML
- Скрипты подключаются в конце body, чтобы DOM был готов

## Структура файлов

- `index.html` - разметка
- `style.scss` - стили (исходник)
- `style.css` - скомпилированные стили
- `index.js` - вся логика поиска и фильтрации
- `course-data.js` - данные с курсами
- `img/` - картинки курсов

## Как это работает

1. При загрузке страницы course-data.js создаёт window.courses с массивом курсов
2. index.js читает эти данные и сортирует их по категориям
3. Рендерит кнопки фильтров и кaрточи курсов
4. При клике на категорию или вводе в поиск - перерисовывает только нужные карточки
5. Если ничего не найдено - показывается сообщение "Ничего не найдено"
